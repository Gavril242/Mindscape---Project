# Welcome mindscape

## Project info
 In order to run this project uou simply run this followring command:
 
 
```bash
 npm install

```
```bash
 npm run dev
```
I personally used Node.js version 10.9.0

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
